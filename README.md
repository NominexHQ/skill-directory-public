# Nominex Skill Directory (Public)

Public catalog of Nominex plugins and skills available for Claude Code.

## Installation

To add this marketplace to your Claude Code installation:

```bash
claude marketplace add https://github.com/NominexHQ/skill-directory-public
```

Then install individual plugins:

```bash
claude plugin install pmm
```

## Available Plugins

See `marketplace.json` for the full catalog with descriptions and skill listings.
